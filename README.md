# cf-wrap-gen
Cloudflare Warp WireGuard Config Generator

This project provides a simple, browser-based tool to generate WireGuard configuration file for connecting to Cloudflare Warp.  It automates the process of creating the necessary keys, registering with Cloudflare's API, and retrieving the connection information.
